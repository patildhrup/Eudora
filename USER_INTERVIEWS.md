# User interviews

## Target personas

1. **Startup CTO** — 5–30 eng, owns tool budget
2. **VP Engineering** — 30–100 eng, Copilot + Cursor sprawl
3. **Founder/CEO** — pre-PMF, personal ChatGPT + API experiments

## Script (20 min)

1. Which AI tools do you pay for today? Monthly rough spend?
2. Who approves new AI subscriptions?
3. Ever discovered duplicate tools on the team?
4. Would you trust a calculator that shows list-price savings? What proof do you need?
5. What would make you book a call with Credex?

## Hypotheses to validate

| Hypothesis | Signal |
|------------|--------|
| Teams underestimate overlap | Names 3+ chat tools |
| Seat sprawl is common | Copilot seats > active devs |
| $500/mo is consultation threshold | "I'd talk to someone" at that number |
| Share links useful internally | Wants to send to CFO |
